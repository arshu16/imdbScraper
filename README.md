# imdbScraper
Script that gets the top 10 most popular titles from imdb in an year range


# Returned structre

CLI Options: 
``` python Crawler.py *startYear* *endYear* *genres* ```

```
data = {
  title: { 
          synopsis: String
          genre: Array
          imgLink: String
          rating: Float
          link: String
         },
         ....
     }
```
